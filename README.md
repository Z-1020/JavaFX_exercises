# JavaFX_exercises
Ce dépôt contient les travaux pratiques de JavaFX effectués durant la première année du BUT Informatique.
# Guide pour installer et importer un projet Maven dans Eclipse

1. **Installer Eclipse**  
   Téléchargez et installez Eclipse IDE pour Java (ou Java EE selon vos besoins).

2. **Cloner le projet**  
   Clonez le projet depuis le dépôt Git sur votre ordinateur.

3. **Créer un dossier pour le projet**  
   Créez un dossier où vous souhaitez installer votre projet Eclipse.

4. **Choisir l’espace de travail dans Eclipse**  
   Lors du lancement d’Eclipse, sélectionnez le dossier créé comme **workspace**.

5. **Importer le projet Maven**  
   - Dans le menu **File**, cliquez sur **Import…**  
   - Sélectionnez **Maven → Existing Maven Projects**, puis cliquez sur **Next**.

6. **Sélectionner le projet**  
   Parcourez votre disque et sélectionnez le dossier nommé `gribouille`.  
   Eclipse détectera automatiquement le fichier `pom.xml`.

7. **Finaliser l’importation**  
   Cliquez sur **Finish**.  
   Le projet Maven sera maintenant importé et prêt à être utilisé.
   
# Exécuter le projet

1. Ouvrez le fichier `App.java` dans le projet.
2. Cliquez sur la flèche verte en haut à droite pour lancer l’exécution.
3. L'application devrait maintenant démarrer et être opérationnelle.
